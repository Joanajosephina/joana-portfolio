Automation

4 n8n workflows — fully automated

Every handoff between systems is automated. No manual status updates, no missed experiments.

1

Idea Library → Experiment Backlog

Trigger: "Move to Backlog" checkbox = true. Auto-creates Backlog entry with mapped fields (Hypothesis, Funnel Stage, Status = Idea). Polling: every 1 min.

Notion TriggerFilter nodeHTTP POST /v1/pages
2

Backlog Approved → Active Experiments

Trigger: Status = "Approved". One automation creates the active experiment record, while the intake validation layer ensures required launch fields are complete before the experiment can move live.

Notion TriggerValidation layerHTTP POST /v1/pages
3

Intake Validator + Email

Trigger: any update to Active Experiments. If-node: checks 8 required fields (Hypothesis, Primary KPI, Funnel Stage, Review Date, Status not in terminal states). True: Status → "Ready to Launch". False: Status → "Needs Info" + Gmail email to owner with missing fields.

If node (8 conditions)Notion UpdateGmail OAuth2Google Cloud Console
4

GrowthBook → Notion Sync

Schedule trigger (every 5 min). GET GrowthBook API /api/v1/experiments. Code node extracts experiment ID, name, status. POST Notion database query to find matching entry by name. PATCH Notion page: Status = "Running", GrowthBook ID, Start Date = now.

Schedule TriggerGrowthBook APICode (JS)Notion query + patch
